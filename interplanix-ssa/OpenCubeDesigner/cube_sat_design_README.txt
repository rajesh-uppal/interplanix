# FreeCAD CubeSat Model (.FCStd)

This file is intended to be opened with [FreeCAD](https://www.freecad.org/). It includes a parametric 1U CubeSat model with:

- Payload compartment
- Solar panel mounts
- Battery bay and antenna slots

To create this model:
1. Use the Part Design workbench.
2. Create a 100x100x100mm outer cube.
3. Use sketches and padding to define internal bays.
4. Export `.STEP` or `.STL` for manufacturing or simulation.
