🧪 2. How It Works

  
This page calculates orbital parameters based on your input:

    • 
    
  • ECC = Eccentricity (0 = circular)
    • 
    
  • INC = Inclination (0 = equator, 90 = polar)

  • SRP / Drag = Only included for Earth orbits (not modeled around Moon)
